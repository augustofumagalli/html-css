Capítulo 2 Aula 1 – Como a Internet funciona?

	01000001 = A (ondas quadradas)
	0 e 1 são bits e o conjunto de 8 bits se torna um byte
	Código multibyte UTF-8?
	MB (megabyte) é diferente de Mb (Megabits)
	MB é usado para armazenamento e o Mb é para transmissão (internet, por exemplo)
	O aparelho de internet serve para fazer a modulação das ondas quadradas que o computador entende para ondas senoidais que a rede de comuniçação entende
	MOdulação - Transformação de ondas quadradas em senoidais / DEModulação - transformação de ondas senoidais em ondas quadradas
	DNS - local de nomes do servidor

Capítulo 3 Aula 1 – A diferença entre HTML, CSS e JavaScript
	
	Vc n programa em HTML e CSS, vc desenvolve 
	HTML = Hypertext markup language (Linguagem de marcação para hipertexto)
	CSS = Cascading Style sheets (Folhas de estilo em cascata)
	A linguagem HTML é focada em CONTEÚDO
	CSS é foco em design
	JavaScript = interatividade
CONTEÚDO EM HTML
	<h1> (ABERTURA DE TAG) EXEMPLO DE TÍTULO </h1> (FECHAMENTO DE TAG)
	<p> EXEMPLO DE PARÁGRAFO </p>
ESTILO EM CSS
	h1{
	   font-family: Arial;
	   font-size: 20pt;
	   color: blue;
	  }
ESTRUTURA BÁSICA DE DOCUMENTO HTML 
	<!DOCTYPE html>
	<html lang="pt-br">
	   <head> (PARTE DE CIMA DO NAVEGADOR)
	       <meta charset="UTF-8"> (O meu site vai ser compatível com acentuação - português)
	       <meta name="viewport" 
	        content="width=device-width,inicial-scale=1.0">
	       <title>Document</title>
	   <head>
	   <body>
	       	<h1>Olá, Mundo!</h1> (TÍTULO)
	        <p>Estou muito feliz</p> (PARÁGRAFO)

	   <body>
	<html>
